## 🔧 Cara Melihat Solusi

1. Pilih folder problem berdasarkan nomor LeetCode
2. Baca `README.md` untuk memahami problem statement
3. Lihat file solusi (`.cpp`, `.js`, atau `.go`)

```bash

cat README.md        # Baca problem description
cat two-sum.cpp      # Lihat solusi
```
